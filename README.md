# 🚀 Run Locally

## 1. Clone the repository

```bash
git clone https://github.com/sumon-webs/SwiftCart.git
```

## 2. Navigate to the project directory

```bash
cd SwiftCart
```

## 3. Install dependencies

```bash
npm install
```

## 4. Configure Environment Variables

If your project requires environment variables, create a `.env.local` file and add the necessary values.

```env
NEXT_PUBLIC_API_URL=
```

> Skip this step if your project doesn't use environment variables.

## 5. Run the development server

```bash
npm run dev
```

Open **http://localhost:3000** in your browser to view the application.
