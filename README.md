# SwiftCart

Welcome to **SwiftCart**, a modern, high-performance e-commerce web application. This project is designed to provide a seamless shopping experience with a clean and responsive user interface.

## 🚀 Live Demo
https://swift-cart-black.vercel.app/

## 🛠 Tech Stack
This project leverages the power of modern web technologies:
- **Next.js**: A powerful React framework for fast and SEO-friendly web applications.
- **Tailwind CSS**: A utility-first CSS framework for rapid UI development and styling.
- **HeroUI**: A beautiful, fast, and accessible UI component library for better user experience.
- **Vercel**: Deployed on the Vercel platform for optimal performance and scalability.

## ✨ Key Features
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
- **Modern UI**: Clean and intuitive interface components using HeroUI.
- **Performance**: Built with Next.js to ensure fast load times and smooth navigation.
- **Clean Code**: Maintained with a structured and scalable codebase.
